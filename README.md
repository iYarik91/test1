# Dockerized Flask APP - test1 (Hello, Docker world!)

## Quick start

  git clone https://github.com/iYarik91/test1.git && cd test1
  
  terraform init
  terraform plan
  terraform apply
  
  # Wait until infrastructure become ready
  
  ansible-playbook main.yaml
  
  # You are good to go
  
  http://<hostname>:5000/
